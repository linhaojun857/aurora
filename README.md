## 相关技术

### 前端

前台：vue3 + pinia + vue-router + axios

后台：vue2 + vuex + vue-router + axios + element-ui 

### 后端

springboot + springsecurity + mysql + redis + mybatisplus + quartz + rabbitmq + elasticsearch + docker + niginx 

同时还集成了QQ登录
